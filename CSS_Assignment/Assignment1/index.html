Sample demo demonstrating position, float, display and visibility concepts.

Positioning Properties:

1. Static=>

All HTML elements are positioned static by default. Elements render in order, as they appear in the document flow.
The properties are left, right, top, bottom, and z-index do not affect the elements positioned as static.

2. Relative=>

It is used when we need to position an HTML element relative to its normal position.
The element is positioned relative to its normal position, so "left:20px" adds 20 pixels to the element's LEFT position

3. Absolute=>

An element with the absolute position will move according to the position of its parent element.
The element is positioned relative to its first positioned (not static) ancestor element

4. Fixed=>

An element with position:fixed will remain stuck to a specific position even after the page is scrolled.
The element is positioned relative to the browser window

5. Sticky=>

FLoat Properties=>

1. Left--> The elements floats to the left of its container.
2. Right--> The elements floats to the right of its container.
3. None--> The element does not float (it will be displayed just where it occurs in the text). This is default.
4. Inherit--> The element inherits the float value of its parent.

Display Properties:

1. inline -->The inline element takes the required width only. It doesn't force the line break so the flow of text doesn't break in inline example.
2. inline-block -->The CSS display inline-block element is very similar to inline element but the difference is that you are able to set the width and height.
3. block -->The CSS display block element takes as much as horizontal space as they can. Means the block element takes the full available width. They make a line break before and after them.
4. none-->The "none" value totally removes the element from the page. It will not take any space.
