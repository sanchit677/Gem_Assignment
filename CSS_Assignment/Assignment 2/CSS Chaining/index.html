Chaining Selectors

When writing CSS rules, it's possible to require an HTML element to have two or more CSS selectors at the same time. This is done by combining multiple selectors, which refer to as chaining. For instance, if there was a .chaining class for h1 elements, the CSS would look like

h1.chaining{

}

The code above would select only the h1 elements that have a class of chaining. If a 'p' element also had a class of chaining, the rule in the example would not style the paragraph.
